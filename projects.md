## Recap Insights
**Athlete recap dashboard for Strava + Intervals.icu**

A privacy-conscious recap app that turns your training history into weekly/monthly/yearly insights (volume, consistency, trends). Designed to run lean with minimal server-side state.

### Tech stack
- React + Vite
- Azure Static Web Apps
- Azure Functions (C# isolated)
- Strava OAuth (read)
- Intervals.icu OAuth (read)
- LocalStorage caching
- Cloudflare (domain/DNS)
